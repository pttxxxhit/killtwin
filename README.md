# Automatización de Tareas

Una aplicación móvil desarrollada con Flutter y Python para automatizar tareas comunes de gestión de archivos.

## Características

- Organización automática de archivos por extensión
- Cambio de tamaño de imágenes
- Eliminación de archivos duplicados
- Interfaz intuitiva y fácil de usar

## Tecnologías utilizadas

- Flutter para la interfaz de usuario
- Python para la lógica de procesamiento de archivos
- Flet para la integración de Python con Flutter

## Requisitos

- Flutter SDK
- Python 3.7+
- Android Studio o VS Code con extensiones de Flutter
